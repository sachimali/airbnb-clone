# 🏡 Airbnb Listing Clone

A pixel-perfect frontend clone of the Airbnb Listing Details page, built as part of a frontend engineering assignment. The project recreates the look, feel, and interactions of Airbnb's property listing experience with a strong focus on UI accuracy, accessibility, and responsive design.

## 🌐 Live Demo

**Demo:** https://airbnb-clone-gamma-three.vercel.app/

---

## ✨ Features

- Airbnb-inspired Listing Details page
- Responsive desktop and mobile layouts
- Interactive image gallery
- Photo Tour view
- Lightbox with keyboard navigation
- Sticky booking/reservation card
- Guest information & amenities
- Ratings and reviews section
- Location section with embedded map
- Responsive navigation
- Smooth animations and transitions
- Accessibility support (ARIA, keyboard navigation, reduced motion)

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/<your-username>/airbnb-listing-clone.git
cd airbnb-listing-clone
```

### Install dependencies

```bash
npm install
```

### Run locally

```bash
npm run dev
```

Open:

```
http://localhost:5173
```

## ♿ Accessibility

- Semantic HTML
- Keyboard navigation
- ARIA labels
- Escape key support for modals
- Focus management
- `prefers-reduced-motion` support

---

## ⚠️ Limitations

This project is a frontend UI clone and does not include:

- Backend services
- Authentication
- Payment integration
- Database
- Real booking functionality
- Real-time messaging
- Search across listings
